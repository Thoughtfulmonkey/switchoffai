# Switch Off AI project setup notes

## MKDocs

MKDocs is a static site generator.

1. Download MKDocs from [https://www.mkdocs.org/](https://www.mkdocs.org/).

* Use *mkdocs serve* to create a temporary, dynamically updating static site that's served locally.
* Use *mkdocs build -d "output/path"* to build the static site.

## Vale

Vale is a prose linter that checks for spelling, grammar, and writing style issues.

1. Download Vale from [https://vale.sh/](https://vale.sh/).
2. Run *vale sync* to set up the style folders.
