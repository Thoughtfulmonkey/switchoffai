# DuckDuckGo

A search engine that promises privacy and the protection of personal data.

You can go to the [DuckDuckGo website](https://duckduckgo.com/) to perform a search, or configure your browser to use DuckDuckGo as the default search engine. 

## Using the no AI version

DuckDuckGo provide a version of their search engine with AI-assisted answers turned off and AI-generated images removed from search results. For more information about how DuckDuckGo hide AI-generated images, see [Hiding AI-generated images](#hiding-ai-generated-images).

* To use the no AI version, go to [https://noai.duckduckgo.com/](https://noai.duckduckgo.com/).


## Turning off AI search summaries

DuckDuckGo.com shows AI summaries on the search results page by default. To turn them off, complete the following steps:

1. On the search results page, in the **Search Assist** panel, select the gear icon.
2. In the **How often do you want to see Search Assist answers?** window, select **Never**, and then select **Save**.

## Hiding AI-generated images

To hide AI-generated images in search results, complete the following steps:

!!! note "Note" 
    When hiding AI-generated images, DuckDuckGo blocks all images from a list of sites that are known to host AI-generated content. They don't evaluate individual images. The block list is provided by [uBlockOrigin & uBlacklist Huge AI Blocklist](https://github.com/laylavish/uBlockOrigin-HUGE-AI-Blocklist).

1. Go to [https://duckduckgo.com/settings](https://duckduckgo.com/settings).
2. Turn on **Hide AI-Generated Images**.
