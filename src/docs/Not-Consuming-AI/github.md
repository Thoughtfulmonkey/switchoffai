# GitHub

The developer platform built around Git version control.

## Preventing GitHub from using your data to train AI

By default, GitHub assumes they can use your data to train AI models. To opt-out, complete the following steps:

!!! note "Note" 
    As an alternative to the following steps, go directly to [the Copilot features page](https://github.com/settings/copilot/features), and then disable Github's use of your data.

1. Go to [https://github.com/](https://github.com/), and then sign in.
2. In the upper-right corner, select your profile icon to open the user navigation menu, and then select **Settings**.
3. Scroll down the **Settings page**, and then select **Copilot** in the left-hand menu to expand that section.
4. From the expanded menu, select **Features**.
5. In the **Allow GitHub to use my data for AI model training** panel, select **Disabled** from the list.


## Hiding Copilot

By default, links to Copilot appear in the GitHub UI. To hide Copilot, complete the following steps:

!!! note "Note" 
    As an alternative to the following steps, go directly to [the Copilot features page](https://github.com/settings/copilot/features), and then disable Copilot.

1. Go to [https://github.com/](https://github.com/), and then sign in.
2. In the upper-right corner, select your profile icon to open the user navigation menu, and then select **Settings**.
3. Scroll down the **Settings page**, and then select **Copilot** in the left-hand menu to expand that section.
4. From the expanded menu, select **Features**.
5. In the **Show Copilot** panel, select **Disabled** from the list.
